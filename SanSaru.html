Iteracion n°1

Preparativos

Armar equipos de tres personas...

  • Kikazaru, el mono sordo
  • Mizaru, el mono ciego
  • Iwazaru, el mono mudo


Dinamica

  1. Iwazaru, el mono mudo, debe abandonar la sala...
  2. Kikazaru, el mono sordo, recibe una imagen...
  3. Kikazaru, el mono sordo, le explica sin mostrar la imagen a Mizaru, el mono ciego que ve, el lo escribe...
  4. Kikazaru, el mono sordo, abandona la sala y llama a Iwazaru, el mono mudo para que ingrese....
  5. Mizaru, el mono ciego, entrega lo escrito a Iwazaru, el mono mudo
  6. Mizaru, el mono ciego, dibuja lo que lee...

Iteracion n°2

Preparativos

Armar equipos de dos personas...



Dinamica

  1. los miembros del equipo se sientas de espalda entre ellos...
  2. Uno de los miembros del equipo recibe una imagen...
  3. Describe la imagen al otro miembro de su equipo
  4. El otro miembro, dibuja lo que le explican....
  5. Se pueden hacer preguntas libremente.
